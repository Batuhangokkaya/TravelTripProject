# Seyahat Projesi

ASP.NET MVC (.NET Framework), C#, Entity Framework ve SQL Server ile geliştirilmiş dinamik bir seyahat ve turizm web uygulaması.

## Teknolojiler

- ASP.NET MVC (.NET Framework)
- C#
- Entity Framework
- SQL Server

## Özellikler

- Dinamik destinasyon listesi
- Blog ve blog detayları
- Yorum sistemi
- İletişim sayfası
- Admin paneli
- CRUD işlemleri
